CSS por sus siglas en ingles de Cascading Style Sheets, en español «Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.​

Para realizar las modificaciones en CSS se ocupan diferentes etiquetas lo primero a que se quiere modificar que puede ser un elemento, clase o ID, posteriormente hay que abrir unas llaves"{}" y dentro los que se quiere modificar "color: red;"

Una etiqueta de CSS tiene varios apartados que son los siguientes:

  1. Selector
  2. Propiedad
  3. Valor
